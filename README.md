# Artic API's

The Artic API's is developed to provide API endpoints to the frontend frameworks.

## Setup Using DDEV

To set up the Artic API's project locally using DDEV, follow these steps:

### Prerequisites

- [DDEV](https://ddev.readthedocs.io/en/stable/#installation) installed on your machine.

### Steps

1. Clone the repository to your local machine:

```
git clone <repository_url>
```

2. Navigate to the project directory:
   
```
cd <project_directory>
```

3. Start the DDEV environment:

```
ddev start
```

4. Import the database:

```
ddev import-db
```

5. Access the site:

```
http://<project_name>.ddev.site
```
